# 数据分析作品集

## 超市销售分析项目

### 项目背景
帮助超市找出高销售额和高利润的品类，为经营决策提供依据。

### 核心发现
- 书架销售额最高（230万），是主力品类
- 收纳具利润率最高（27.5%），是隐藏金矿
- 器具销售额第2但利润率最低（9.2%），需优化成本或折扣策略
- 山东销售额最高，西藏最低
- 8月是销售旺季，2月是淡季

### 建议
1. 扶持收纳具（高利润率）
2. 诊断器具（高销售额低利润）
3. 山东经验复制到其他省份
4. 8月旺季提前备货
